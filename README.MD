# ❌⭕ Tic Tac Toe Game

A modern and interactive **Tic Tac Toe Game** built using **HTML, CSS, and JavaScript**.  
The game features a clean UI, smooth gameplay mechanics, player turn tracking, winner detection, and responsive design for an enjoyable gaming experience.

---

# 🚀 Features

- 🎮 Two-player gameplay
- ❌⭕ Dynamic turn switching
- 🏆 Winner detection system
- 🤝 Draw match detection
- 🔄 Restart / Reset game functionality
- ⚡ Fast and responsive UI
- 🎨 Modern and clean interface
- 📱 Responsive design for different screen sizes
- 🖱️ Interactive hover effects
- 💡 Real-time game updates
